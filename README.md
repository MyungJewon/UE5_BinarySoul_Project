# 바이너리 소울 (Binary Soul)

**바이너리 소울**은 선택의 기로에서 발생하는 리스크와 보상을 관리하며 진영 전쟁에 참여하는 **3D 소울라이크 액션 게임**입니다.

---

##  프로젝트 개요
* **장르**: 3D 소울라이크 액션 (Roguelike + Soul-like)
* **플랫폼**: PC (Steam) / 키보드 및 마우스 조작 기반
* **엔진**: Unreal Engine 5 (5.7.2)
* **핵심 키워드**: 선택의 방, 진영 전쟁, 10개의 LED, 리스크/보상
* **개발 기간**: 2025.12.20 ~ (개발중)

---

##  핵심 시스템

### 1. 선택의 방 (Choice Room)
전투 진입 전 플레이어가 거쳐야 하는 의사결정 단계로, 로그라이크적인 무작위 요소가 결합됩니다.
* **동작 메커니즘**: 방 중앙의 빨간색(Red)과 파란색(Blue) 버튼을 통해 [대가]와 [보상]을 선택합니다.
* **선택 기준**: 단순히 색상이 아닌, 현재 플레이어의 HP나 장비 상태에 따라 전략적으로 선택합니다.

| 버튼 타입 | 빨간 버튼 (Red) | 파란 버튼 (Blue) |
| :--- | :--- | :--- |
| **성격** | 확정적/안정적 (Low Risk) | 확률적/극단적 (High Risk) |
| **효과 예시** | 10 데미지 입고 무기 획득 등  | 30% 확률로 전설 아이템 획득 등 |

### 2. 진영 전장 (Faction Battle)
선택한 버튼의 진영 용병이 되어 전장에 참여합니다.
* **진영 로직**: 빨간 버튼 선택 시 빨간 진영이 아군이 되고, 파란 버튼 선택 시 파란 진영이 아군이 됩니다.
* **액션 스타일**: 패링, 회피 등 소울라이크 특유의 묵직한 타격감을 기반으로 합니다.
* **배신 버튼 (Spike)**: 전투 중 이 버튼을 누르면 즉시 진영이 반전되어 적군과 아군이 뒤바뀝니다.

### 3. 10개의 LED 시스템
게임의 전체 흐름을 시각적으로 보여주는 메타 시스템이자 엔딩 결정 조건입니다.
* **점등 조건**: 전투 승리 시 현재 진영의 색상으로 LED가 하나씩 켜지며, 10개가 채워지면 게임이 종료됩니다.
* **엔딩 분기**:
    * **적색 편향 (Red 7+)**: 힘에 의한 강압적 통제 엔딩
    * **청색 편향 (Blue 7+)**: 차가운 이성에 의한 정체 엔딩
    * **균형 (5:5)**: 새로운 공존 혹은 제3의 길 엔딩
